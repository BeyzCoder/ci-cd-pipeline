# CI/CD Pipeline.

This is a project from DEVP-600 class to show how CI/CD can be use in a e-commerce business.

**Must already exist:**

- Dynamodb: `financial-statements`
- ECR: `fastapi-registry`

**Need to change secret key values:**

- AWS_ACCESS_KEY_ID
- AWS_REGION
- AWS_SECRET_ACCESS_KEY
- AWS_SESSION_TOKEN
- ECR_REPOSITORY
- VPC_ID
- VPC_SUBNET_ID
